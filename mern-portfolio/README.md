# MERN Developer Portfolio

A responsive, interactive portfolio website for a Full Stack MERN developer.

## Features

- **Hero** — Particle canvas background, typing animation, animated stat counters
- **About** — Highlight cards with hover effects
- **Skills** — Tabbed panels with animated progress bars
- **Projects** — Category filters with fade transitions
- **Experience** — Animated timeline
- **Contact** — Validated form with loading state
- **Global** — Dark/light theme, scroll reveal, cursor glow, mobile nav, back-to-top

## Quick Start

Open `index.html` in your browser, or run a local server:

```bash
npx serve .
# or
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Customize

1. Edit personal info in `index.html` (name, email, projects, experience)
2. Adjust colors in `css/style.css` (`:root` variables)
3. Change typing phrases in `js/main.js` (`phrases` array)
